☁️ Cloud Sync

Save your inventory to the cloud and access it from any device. No account needed!

📱 How to use on any device:

  1. Save your inventory to cloud (get your Cloud ID)
  2. On any device, go to webstore/admin.html
  3. Click Cloud Sync and enter your Cloud ID
  4. Click "Load from Cloud"
